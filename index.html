Algoritmo 36 "Total de Horas Estudadas na Semana"

Pseudocódigo Visualg:

Var
horas, thoras, mhoras: real
semana: inteiro
Inicio
thoras <- 0
para semana de 1 ate 4 faca
escreva("Digite a quantidade de horas estudadas na semana ", semana, ": ")
leia(horas)
thoras <- thoras + horas
fimpara
escreval
mhoras <- thoras / 4
escreva("Média de horas estudadas por semana: ", mhoras)
escreval
FimAlgoritmo

JavaScript:

const calcularBtn = document.getElementById('calcular');
const mediaElement = document.getElementById('media');
			
calcularBtn.addEventListener('click', () => {
let thoras = 0;
for(let semana = 1; semana <= 4; semana++) {
thoras += parseFloat(prompt(`Digite a quantidade de horas estudadas na semana ${semana}:`));
}
 mediaElement.textContent = `Média de horas estudadas por semana: ${(thoras / 4).toFixed(2)}`;
			});
